Opdracht

Je gaat 3 pagina's namaken met de hulp van Bootstrap, door alleen classes aan te brengen op bestaande elementen.
Alle 3 de pagina's hebben de zelfde HTML maar een andere layout.
Voor ieder puzzel gelden de zelfde technische eisen.
Sommige puzzels hebben ook functionele eisen.

Techinsche eisen

  • Je mag niet de html aanpassen (behalve bij het toevoegen van classes)
  • Je mag geen eigen css/style inbrengen (d.m.v. een file of inline style)
